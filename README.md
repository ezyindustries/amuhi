# AMUHI OS - Digital Association Platform

Platform Digital Asosiasi Travel Umrah sebagai "Operating System (OS)" industri.

## Fitur Utama (MVP Phase 1)

1.  **Member Dashboard**: Pusat aktivitas anggota dengan status membership, risiko industri, dan notifikasi.
2.  **Regulation Center**: Single source of truth untuk regulasi Kemenag & Saudi (Placeholder).
3.  **SOP Library**: Akses dokumen standar operasional (Placeholder).
4.  **Discussion Forum**: Forum profesional terkurasi (Placeholder).
5.  **Umrah Academy**: Modul pembelajaran (Placeholder).

## Teknologi

- **Frontend**: React (Vite)
- **Styling**: Vanilla CSS (Variables, Flexbox/Grid, Glassmorphism)
- **Icons**: Lucide React
- **Routing**: React Router DOM

## Cara Menjalankan

1.  Install dependencies:
    ```bash
    npm install
    ```
2.  Jalankan server development:
    ```bash
    npm run dev
    ```
3.  Buka browser di `http://localhost:5173`

## Struktur Project

- `src/components/layout`: Komponen tata letak (Sidebar, Header).
- `src/pages`: Halaman utama (Dashboard).
- `src/index.css`: Design system global (Warna, Typography, Utilities).
